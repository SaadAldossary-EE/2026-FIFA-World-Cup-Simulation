# 🎲 Monte Carlo Simulation

The tournament engine runs repeated probabilistic simulations to estimate:

* Match outcomes
* Knockout progression
* Champion probabilities
* Upset scenarios

This allows the simulator to generate dynamic and realistic FIFA World Cup tournaments.

---

# 🏟️ Tournament Engine

The simulator includes:

* Round of 32
* Round of 16
* Quarterfinals
* Semifinals
* Third Place Match
* Final

Including:

* Penalty shootouts
* Score generation
* Dynamic advancement logic

## ▶️ How To Run Locally

Clone the repository:

```bash
git clone https://github.com/SaadAldossary-EE/2026-FIFA-World-Cup-Simulation.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Streamlit app:

```bash
streamlit run app.py
```
